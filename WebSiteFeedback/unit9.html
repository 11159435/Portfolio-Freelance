Unit 9 – Spreadsheets

This unit covered the practical uses of spreadsheets in real-life scenarios, focusing on both the basics and advanced features of spreadsheet software. During the unit, we explored programs like Microsoft Excel and Google Sheets, learning how to create, edit, and format spreadsheets for a variety of purposes—from simple lists to complex data analysis.

We learned how to enter and organize large sets of data efficiently. The curriculum emphasized the use of formulas and functions to perform calculations automatically, such as SUM, AVERAGE, COUNT, and logical functions like IF and VLOOKUP. We practiced building formulas that could adapt to changing data and using functions to extract meaningful insights from raw information.

A major part of the unit was dedicated to data visualization: we created charts, graphs, and tables to present data clearly for different audiences. We applied conditional formatting to highlight key trends and anomalies, making spreadsheets more informative and interactive. Lessons also included sorting and filtering data to quickly find the information needed.

The unit included real-world applications such as managing personal or business budgets, tracking expenses, scheduling, and analyzing survey results. We explored advanced tools like pivot tables for summarizing data and practiced protecting sensitive information using spreadsheet security features.

Skills gained: Through this unit, I became confident in using spreadsheets for data organization, analysis, and presentation. I developed strong skills in automating calculations, visualizing data, and applying advanced features like pivot tables. These abilities help me solve practical problems and work efficiently with data in both academic and professional settings.
